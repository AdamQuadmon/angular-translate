$translate (service in module translate )

Description

The $translate service is the actual core of angular-translate. It excepts a translation id and optional interpolate parameters to translate contents.

 $scope.translatedText = $translate('HEADLINE_TEXT');

Dependencies

  • $interpolate
  • $log
  • $rootScope
  • $q

Usage

$translate(translationId[, interpolateParams]);

Parameters

  • translationId – {string} –

    A token which represents a translation id

  • interpolateParams(optional) – {object=} –

    An object hash for dynamic values

Methods

  • fallbackLanguage()

    Returns the language key for the fallback language.

    Returns

    {string} –

    fallback language key

  • preferredLanguage()

    Returns the language key for the preferred language.

    Returns

    {string} –

    preferred language key

  • storage()

    Returns registered storage.

    Returns

    {object} –

    Storage

  • storageKey()

    Returns the key for the storage.

    Returns

    {string} –

    storage key

  • uses(key)

    Tells angular-translate which language to uses by given language key. This method is used to change language at runtime. It also takes care of storing the language key in a configured store to let your app remember the choosed language.

    When trying to 'use' a language which isn't available it tries to load it asynchronously with registered loaders.

    Parameters

    • key – {string} –

      Language key

    Returns

    {string} –

    Language key
