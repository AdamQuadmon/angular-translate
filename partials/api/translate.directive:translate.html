translate (directive in module translate )

Description

Translates given translation id either through attribute or DOM content. Internally it uses translate filter to translate translation id. It possible to pass an optional values object literal as string into translation id.

Dependencies

  • $filter
  • $interpolate

Usage

as attribute
<ANY translate="{string}"
     [values="{string}"]>
   ...
</ANY>

Parameters

  • translate(optional) – {string=} –

    Translation id which could be either string or interpolated string.

  • values(optional) – {string=} –

    Values to pass into translation id. Can be passed as object literal string or interpolated object.
