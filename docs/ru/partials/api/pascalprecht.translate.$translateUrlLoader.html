$translateUrlLoader (service in module pascalprecht.translate )

Description

Creates a loading function for a typical dynamic url pattern: "locale.php?lang=enUS", "locale.php?lang=deDE", etc. Prefixing the specified url, the current requested, language id will be applied with "?lang={key}". Using this service, the response of these urls must be an object of key-value pairs.

Dependencies

  • $q
  • $http

Usage

$translateUrlLoader(options);

Parameters

  • options – {object} –

    Options object, which gets the url and key.
