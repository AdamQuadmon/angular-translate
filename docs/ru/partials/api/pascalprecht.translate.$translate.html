$translate (service in module pascalprecht.translate )

Description

The $translate service is the actual core of angular-translate. It excepts a translation id and optional interpolate parameters to translate contents.

 $scope.translatedText = $translate('HEADLINE_TEXT');

Dependencies

  • $interpolate
  • $log
  • $rootScope
  • $q

Usage

$translate(translationId[, interpolateParams]);

Parameters

  • translationId – {string} –

    A token which represents a translation id

  • interpolateParams(optional) – {object=} –

    An object hash for dynamic values

Methods

  • fallbackLanguage()

    Returns the language key for the fallback language.

    Returns

    {string} –

    fallback language key

  • preferredLanguage()

    Returns the language key for the preferred language.

    Returns

    {string} –

    preferred language key

  • proposedLanguage()

    Returns the language key of language that is currently loaded asynchronously.

    Returns

    {string} –

    language key

  • refresh(lankKey)

    Refreshes a translation table pointed by the given langKey. If langKey is not specified, the module will drop all existent translation tables and load new version of those which are currently in use.

    Refresh means that the module will drop target translation table and try to load it again.

    In case there are no loaders registered the refresh() method will throw an Error.

    If the module is able to refresh translation tables refresh() method will broadcast $translateRefreshStart and $translateRefreshEnd events.

    Parameters

    • lankKey – {string} –

      A language key of the table, which has to be refreshed

    Returns

    {promise} –

    Promise, which will be resolved in case a translation tables refreshing process is finished successfully, and reject if not.

    Example

    // this will drop all currently existent translation tables and reload those which are // currently in use $translate.refresh(); // this will refresh a translation table for the enUS language $translate.refresh('enUS');

  • storage()

    Returns registered storage.

    Returns

    {object} –

    Storage

  • storageKey()

    Returns the key for the storage.

    Returns

    {string} –

    storage key

  • uses(key)

    Tells angular-translate which language to uses by given language key. This method is used to change language at runtime. It also takes care of storing the language key in a configured store to let your app remember the choosed language.

    When trying to 'use' a language which isn't available it tries to load it asynchronously with registered loaders.

    Returns promise object with loaded language file data

    Parameters

    • key – {string} –

      Language key

    Returns

    {string} –

    Language key

    Example

    $translate.uses("en_US").then(function(data){ $scope.text = $translate("HELLO"); });
