$translateStaticFilesLoader (service in module pascalprecht.translate )

Description

Creates a loading function for a typical static file url pattern: "lang-enUS.json", "lang-deDE.json", etc. Using this builder, the response of these urls must be an object of key-value pairs.

Dependencies

  • $q
  • $http

Usage

$translateStaticFilesLoader(options);

Parameters

  • options – {object} –

    Options object, which gets prefix, suffix and key.
